Structorizer ist ein kleines, leicht intuitiv nutzbares Tool zum Erstellen 
von Nassi-Shneiderman-Diagrammen (NSD), einem Diagrammtyp zur Darstellung 
von Programmentwuerfen im Rahmen der strukturierten Programmierung (Struktogramme) 

Structorizer 3.18 von http://structorizer.fisch.lu aus dem Verweis  structorizer 3.18
oder moderner laedt eine File structorizer_3_18.zip herunter, welches in einem lokalen
Verzeichnis entpackt werden muss. 

Alternativ kann Structorizer auch als JAVA Web Start - Projekt oder als
JAVA-Applet gestartet werden.

Das Unterverzeichnis dist enthaelt Structorizer.exe, welches sofort aufgerufen
werden kann.

Structorizer kann an die deutsche Sprache und angepasst und mit spezifischen
Schluesselwoertern ausgestattet werden (Einstellungen | Strukturen) und
(Einstellungen | Parser). Beispielsweise kann in die Sprache C ausgegeben werden,
dazu sollten im Struktogramm die fuer C typischen Ein- und Ausgabefunktionen
eingestellt werden.

Unter Datei | Exportieren | Quellcode koennen unterschiedliche Ziel-Programmier-
sprachen ausgewaehlt werden, unter Datei | Import kann von PASCAL importiert
werden. Unter Datei | Exportieren | Bild koennen unterschiedliche Grafikformate
fuer die Ausgabe ausgewaehlt werden, u.a. das png - oder jpg - Format.
